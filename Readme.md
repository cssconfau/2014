# CSSConf Australia 2014 Site

http://2014.cssconf.com.au/

## Dependencies

* Node
* Sass

## Installing

* `npm install -g grunt-cli`
* `npm install`
* `gem install bundle`
* `bundle install`

## Building

* `grunt dev`
* `open http://localhost:3333/`

## Releasing

* `grunt release`
